アイコンの設定
説明

Java Swingではアイコンはデフォルトのままだとコーヒーカップの絵になっています。
これを任意のアイコンに変更します。

画像ファイルのURLを取得し、このURLからImageインスタンスを生成、JFrameのsetIconImage()で設定という手順となります。

アイコンに設定する画像ファイルは任意の場所を指定出来ますが、例ではjarに固めた際のJFrameのclassの位置から同じjarファイル内に存在するgifファイルのURLを取得しています。

サンプルソース : src_icon.zip
サンプルアプリ : sample_icon.jar
コード例
 

    java.net.URL url = MainClass.class.getResource("/img/star.gif");
    this.setIconImage(java.awt.Toolkit.getDefaultToolkit().createImage(url));
