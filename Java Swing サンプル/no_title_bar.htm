タイトルバーの非表示
説明

タイトルバーを表示させないようにするためには、JFrame.setUndecorated()を使用します。
本来の意味は装飾の有効化と無効化を切り替えるということであり、パラメータをtrueにすると装飾が無効化されます。

この場合の装飾とはフレーム（枠）とタイトルバーを指し、タイトルバー以外にも枠も消えています。

サンプルソース : src_no_title_bar.zip
サンプルアプリ : sample_no_title_bar.jar
コード例
 
   jFrame.setUndecorated(true);
画面中央に表示
画面中央に表示させるためには、JFrame.setLocationRelativeTo()を使用します。
パラメータにはJFrame表示させるのに基準となるコンポーネントを指定します。
これをnullとするとJFrameは画面中央に表示されます。 
    jFrame.setLocationRelativeTo(null);
タイトルバーの非表示、画面中央への表示ともコード例はJFrameですが、JDialogにも使用可。
オープニングダイアログに使用すると良いかも。
