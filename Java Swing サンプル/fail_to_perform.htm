.jarファイルが実行出来ない場合
説明
通常は.jarファイルに同梱のMANIFEST.MF内に以下のようにメインクラスを記述すればファイルは実行可能となります。
※パッケージsample内のMainFrameクラスがメインクラスの場合。 
Main-Class: sample.MainFrame

しかし、コマンドプロンプトから、
javaw -jar xxx.jar

で実行可能なのにダブルクリックでは
"Could not find the main class: sample.MainFrame. Program will exit."

というエラーメッセージが表示され実行出来ない場合、ファイルの関連付けが誤っている可能性があります。

関連付けの変更
フォルダオプションから
「ファイルの種類」タブ →
一覧から拡張子がJARの行を選択 →
「詳細設定」ボタン →
「編集」ボタン →
アクションを実行するアプリケーション(L)
を変更します。

先頭にJAVA_HOMEのパスを記述し、以降

"\javaw.exe" -jar "%1"

を追記します。

ファイルの関連付けが原因であれば、これで実行可能になります。

これは、ダブルクリックで実行されるプログラムはあくまでもファイルに関連付けされたものが選択されることによるものです。
